Printer guide — test your duplex setting, or print without a duplexer

Which flip setting does my printer use?

  1. Download the one-sheet test file and print it double-sided, landscape, actual size.
  2. Pick up the printed sheet and turn it over like a book page (around its left or right edge).
  3. One sentence on the back will read the right way up — it names the duplex setting to choose above. You only need to do this once per printer.

Test a full booklet — folding, sheet order, and backs-first printers

The one-sheet test can’t catch a printer that pairs backs with the wrong fronts (backs-first stacking) — that needs several sheets. This downloads a 12-page test booklet using the form, direction, and duplex settings you’ve chosen above, imposed exactly like a real document:

  1. Every page says which sheet and side it must land on — if the two sides of any printed sheet show different sheet numbers, tick “Backs in reverse order” (Manual mode) and print again.
  2. The cover page carries step-by-step folding and stacking instructions for the chosen form — nested single signature or independent quires — in the chosen direction.
  3. If the finished booklet reads 1 to 12 in order, your settings are right: save them under the printer’s name above.

My printer can’t print double-sided

Set “Double-sided printing” above to Manual — you’ll get two files instead of one:

  1. Print the fronts file.
  2. Take the printed stack and put it back in the paper tray. Which way up depends on your printer — run the test sheet through these same steps first (print its page 1, reinsert, print its page 2) and check which sentence comes out upright and on the same sheet.
  3. Print the backs file.

If your printer stacks output face up, the backs will pair with the wrong fronts — tick “Backs in reverse order” and print again. Whichever sentence read upright in your test is also the “printer flip setting” to pick above. Once it works, type a name in the Printer box and press Save — this browser will remember the settings for that printer.
