Printer guide — test your duplex setting, or print without a duplexer

Which flip setting does my printer use?

  1. Download the one-sheet test file and print it double-sided, landscape, actual size.
  2. Pick up the printed sheet and turn it over like a book page (around its left or right edge).
  3. One sentence on the back will read the right way up — it names the duplex setting to choose above. You only need to do this once per printer.

My printer can’t print double-sided

You can duplex by hand using odd/even pages — the booklet file alternates front, back, front, back:

  1. In the print dialog, print odd pages only (these are the sheet fronts).
  2. Take the printed stack and put it back in the paper tray. Which way up — and whether the order must be reversed — depends on your printer, so run the test sheet through these same two steps first: print its page 1, reinsert, print even pages only, and check which sentence is upright and that it’s on the same sheet.
  3. Print the booklet the same way: odd pages, reinsert exactly as in your test, then even pages.

Whichever sentence read upright in your test is also the “printer duplex setting” to pick above.
