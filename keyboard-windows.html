LigaCheh for Windows

An MSKLC keyboard layout.

A Lisān ud-Daʿwat keyboard layout for Windows, authored with the Microsoft Keyboard Layout Creator (MSKLC).

Install

  • Download the layout source: maqalaAra.klc
  • Open it in MSKLC (File → Load Source File…), then Project → Build DLL and Setup Package.
  • Run the generated setup.exe, then add the layout under Settings → Time & language → Language & region.
