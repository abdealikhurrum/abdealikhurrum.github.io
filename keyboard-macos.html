LigaCheh Keyboard · macOS

LigaCheh for macOS — layout

A Ukelele keyboard layout.

A Lisān ud-Daʿwat keyboard layout for macOS, built with Ukelele. Quick to install and works in every app.

Install

  • Download and unzip lsdMac-ukelele.zip.
  • Move lsdMac.bundle into ~/Library/Keyboard Layouts/ (or /Library/Keyboard Layouts/ for all users).
  • Log out and back in, then add it under System Settings → Keyboard → Input Sources → +.
