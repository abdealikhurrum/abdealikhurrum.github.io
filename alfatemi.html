Typeface

AlFatemi

A display typeface for headings and titles.

لسان الدعوة

AlFatemi is a companion display face to FatemiMaqala, shaped for titles, headings, and short Arabic-script display text.

Get the font

Download .ttf Download .otf Install on iPhone / iPad
  • Mac / Windows: open the downloaded font file and click Install.
  • iPhone / iPad: tap “Install on iPhone / iPad” to download the configuration profile, then go to Settings → Profile Downloaded → Install.
