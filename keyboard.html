LigaCheh Keyboard

Type Lisān ud-Daʿwat everywhere.

A purpose-built keyboard for Lisān ud-Daʿwat — the Arabic-script language of the Dawoodi Bohra community — with layouts that also cover standard Arabic and Urdu. Pick your platform below.

iOS & iPadOS

The LigaCheh keyboard app, with smart suggestions and a learning mode.

Android

The LSDKeyboard input method for Android phones and tablets.

Windows

An MSKLC keyboard layout for Windows.

macOS — layout

A Ukelele keyboard layout, installed in seconds.

macOS — Input Method

A richer macOS input method (in progress).

More platforms

Cross-platform builds via Keyman, plus a layout for the Unihertz Titan 2 physical keyboard:

  • Keyman package: lsd.kmp · lisanflex.kmp (install with Keyman on Windows, macOS, Linux, iOS, or Android)
  • Pastiera layout for the Unihertz Titan 2: PastieraKeyboard-ArabicLSD.json

The LigaCheh keyboard stores nothing off your device — see the site-wide Privacy Policy.
