Typeface

FatemiMaqala

A naskh-style typeface for Lisān ud-Daʿwat.

كيم چھو

FatemiMaqala is a text typeface for Lisān ud-Daʿwat and Arabic-script writing, with the extended letters and diacritics used in Urdu and Persian. It is the face used across the LigaCheh keyboard and the ashaar.js tools.

Get the font

Download .ttf Install on iPhone / iPad
  • Mac / Windows: open the downloaded .ttf and click Install.
  • iPhone / iPad: tap “Install on iPhone / iPad” to download the configuration profile, then go to Settings → Profile Downloaded → Install. The profile is signed, so iOS shows it as Verified.
